# Tic_tac_game
Introduction

Tic Tac Toe is a classic two-player game that involves trying to create a line of three of your own symbols (traditionally "X" or "O") on a 3x3 grid. The simplicity of the game makes it a popular choice for educational purposes and beginner programming projects.

Objective
The primary goal of Tic Tac Toe is to align three of your symbols horizontally, vertically, or diagonally before your opponent does. The game ends when one player achieves this objective, or if the entire grid is filled without a winner, resulting in a draw.

Technologies Used

HTML: Markup for the game board and interface.
CSS: Styling to enhance the visual appeal.
JavaScript: Implementation of game logic and interactivity.
Features

Grid: A 3x3 grid where players make their moves.
Player Turns: Players take turns placing their symbols on empty grid spaces.
Winning Condition: The game checks for a winning combination after each move.
Draw Condition: If the grid is filled without a winner, the game ends in a draw.
Restart Option: Players can restart the game to play again.
